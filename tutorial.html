KrakenPrime Configuration Guide

Everything you need to get Kraken Prime up and farming — from first launch and LDPlayer setup to deployment presets and auto-upgrade. Follow along with the video walkthroughs below.

1 · First Run 2 · LDPlayer & ADB 3 · Deployment Presets 4 · Farming Settings 5 · Auto-Upgrade 6 · Start Farming
1

First Run & Setup

Download KrakenPrime.exe from the home page and run it — there's no installer. On first launch Windows SmartScreen may warn you; click More info → Run anyway.

  1. Launch KrakenPrime.exe.
  2. Work through the setup-readiness checklist on the main screen.
  3. Green checks mean you're ready; anything red points to what still needs fixing.

LDPlayer & ADB Setup

Kraken Prime drives Clash of Clans inside LDPlayer over ADB. Configure the emulator once and you're set.

  • Set the LDPlayer resolution to 1600 × 900.
  • Enable ADB debugging in LDPlayer settings.
  • Open Clash of Clans and make sure it's fully loaded before starting the bot.

Deployment Presets

Use the visual deployment planner to lay out how troops drop, then save it as a reusable preset.

  1. Open the deployment planner.
  2. Place your troops / spells on the layout to define the attack.
  3. Save it as a named preset so Kraken Prime deploys it every cycle.

Farming Settings

Tune how Kraken Prime scouts and attacks so it only spends troops on bases worth hitting.

  • Set your minimum loot thresholds (gold / elixir / dark elixir).
  • Choose the deployment preset to use for attacks.
  • Configure the attack cycle and how it scouts for loot.

5

Auto-Upgrade

Turn on opportunistic auto-upgrade so idle loot goes straight into walls and buildings while the bot runs.

  • Enable auto-upgrade for walls and/or buildings.
  • Set priorities so the right structures get upgraded first.
  • Kraken Prime spends spare resources between attack cycles automatically.

6

Start Farming

With everything green on the checklist, start the bot and watch the live preview to confirm it's behaving as expected.

  1. Confirm the readiness checklist is all green.
  2. Press Start and monitor the live preview.
  3. Leave it running — Kraken Prime farms and upgrades on autopilot.
